개인정보처리방침

최종 수정일: 2026년 3월

📢 이 확장 프로그램은 사용자의 개인정보를 수집하거나 외부 서버로 전송하지 않습니다.

1. 개요

"D2R 아이템 식별기"(이하 "확장 프로그램")는 디아블로2 레저렉션 아이템 스크린샷을 분석하여 Traderie의 시세를 조회하는 Chrome 확장 프로그램입니다. 본 방침은 확장 프로그램이 처리하는 데이터의 범위와 방식을 설명합니다.

2. 수집하는 데이터

이 확장 프로그램은 다음 데이터를 처리합니다:

  • 스크린샷 이미지 — 사용자가 직접 업로드하거나 붙여넣은 이미지. OCR 처리는 사용자 PC에서 완전히 로컬로 수행되며, 이미지는 어떠한 외부 서버에도 전송되지 않습니다.
  • 즐겨찾기 데이터 — 아이템 정보 및 가격 데이터. Chrome의 로컬 스토리지(chrome.storage.local)에만 저장되며, 사용자 기기 밖으로 전송되지 않습니다.
  • 설정값 — 게임 모드(래더/스탠, 소프트/하드) 등 사용자 설정. 마찬가지로 로컬에만 저장됩니다.

3. 외부 API 연동

아이템 시세 조회를 위해 아래 두 사이트의 공개 API를 사용합니다:

  • Traderie (traderie.com)
    아이템 판매 중인 가격 데이터 조회에 사용됩니다. 해당 사이트 운영자로부터 사전 허가를 받았습니다.
    아이템 거래완료 이력은 해당 사이트의 운영정책을 위반하기 때문에 제공하지 않습니다.
    트레더리의 거래방법 및 회원가입은 트레더리 이용수칙을 따르십시오.
  • D2Emu (d2emu.com)
    아이템 기초 데이터(옵션, 분류 등) 조회에 사용됩니다. 해당 사이트 운영자로부터 API 사용에 대한 사전 허가를 받았습니다.

이 API 호출 시 아이템 검색 파라미터(아이템 종류, 게임 모드 등 비식별 정보)만 전송됩니다. 사용자의 개인 식별 정보, 계정 정보, 위치 정보 등은 전혀 포함되지 않습니다.

4. 개인정보 미수집 선언

  • 이름, 이메일, 전화번호 등 개인 식별 정보를 수집하지 않습니다.
  • 사용자의 브라우징 기록, 방문 사이트, 검색 기록을 수집하지 않습니다.
  • 수집한 어떠한 데이터도 제3자에게 판매하거나 공유하지 않습니다.
  • 광고 목적으로 데이터를 재사용 하지 않습니다.
  • 별도의 계정 가입이나 로그인을 요구하지 않습니다.

5. 사용 권한 목적

  • storage — 즐겨찾기 및 설정을 사용자 기기에 로컬 저장하기 위해 사용합니다.
  • sidePanel — Chrome 사이드 패널에 확장 프로그램 UI를 표시하기 위해 사용합니다.
  • declarativeNetRequest — Traderie API 요청 시 필요한 HTTP 헤더(Referer)를 설정하기 위해 사용합니다. 사용자 트래픽을 가로채거나 수정하지 않습니다.
  • host_permissions (traderie.com, d2emu.com) — 위 두 사이트의 공개 API에서 아이템 데이터를 가져오기 위해 사용합니다.

6. OCR 처리 방식

아이템 이미지 텍스트 인식(OCR)은 Tesseract.js 라이브러리를 사용하여 사용자의 PC에서 완전히 로컬로 처리됩니다. 업로드된 이미지는 OCR 처리 후 즉시 메모리에서 제거되며, 어떠한 서버에도 전송되지 않습니다.

7. 데이터 보관 및 삭제

즐겨찾기 등 로컬 저장 데이터는 사용자가 Chrome 확장 프로그램을 제거하거나, 확장 프로그램 내 내보내기/삭제 기능을 통해 직접 관리할 수 있습니다.

8. 방침 변경

개인정보처리방침이 변경될 경우, 확장 프로그램 업데이트 노트를 통해 사용자에게 안내합니다. 변경된 방침은 Chrome 웹 스토어 등록 페이지 및 이 문서를 통해 확인할 수 있습니다.

9. 문의

개인정보 처리에 관한 문의 사항은 Chrome 웹 스토어의 지원 탭을 통해 연락하시기 바랍니다.
